# SwiftUI + Agentic AI Roadmap

A 30-day roadmap for iOS engineers covering SwiftUI, iOS architecture, DSA, LLMs, RAG, and agentic AI through interview prep and a practical SwiftMigrate Agent project.

## What this repo contains

This is a static GitHub Pages-ready learning roadmap.

```text
.
├── index.html
├── style.css
├── script.js
└── data
    └── crash-course.json
```

## How it works

- `index.html` contains the page shell.
- `style.css` contains the visual design.
- `script.js` fetches and renders the roadmap.
- `data/crash-course.json` contains all editable roadmap content.

To update the roadmap, edit only:

```text
data/crash-course.json
```

## Suggested GitHub Pages setup

1. Open this repo on GitHub.
2. Go to **Settings**.
3. Go to **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select:
   - Branch: `main`
   - Folder: `/root`
6. Save.

Your site should become available at:

```text
https://anish-raj-mac.github.io/swiftui-agentic-ai-roadmap/
```

## Important safety note
